Comandos de terminal

Comando Descripcion
pwd Imprime la direcci√≥n de la carpeta con la que se esta trabajando
ls Te muestra todos los archivos dentro de la carpeta actual
cd 'carpeta' Cambia la carpeta actual a la que especificaste
cd .. Te mueve un nivel arriba de la jerarquia de carpetas
cd ~ Navega por la carpeta principal de la computadora
mkdir 'carpeta' Crea una nueva carpeta con el nombre dado
rmdir 'carpeta' Elimina una carpeta vacia
rm -rf 'carpeta' A la fuerza elimina una carpeta junto con sus contenidos
touch 'archivo' Crea un nuevo archivo vacio con el nombre dado
cp 'archivo' 'carpeta destinaria' Copia el archivo hacia la carpeta destinaria
mv 'archivo' 'carpeta destinaria' Mueve o renombra el archivo hacia la carpeta destinaria
rm 'archivo' Elimina un archivo
cat 'archivo' Muestra el contenido del archivo en la consola
head 'archivo' Muestra las primeras lineas del archivo
history Te muestra todo el historial de comandos escritos en la terminal
