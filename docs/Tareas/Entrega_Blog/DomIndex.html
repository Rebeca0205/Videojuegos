Preguntas

  1. ¿Qué es el DOM?

    El DOM (Document Object Model) es una representación estructurada del documento HTML en forma de árbol, que permite a los scripts interactuar y modificar el contenido y estructura de la página.

  2. ¿Qué es un nodo?

    Un nodo es cada uno de los elementos que componen el DOM, como etiquetas HTML, atributos, textos, comentarios, etc.

  3. ¿Cómo se accede a un elemento del DOM?

    Se puede acceder a un elemento del DOM usando métodos como getElementById(), getElementsByClassName(), o querySelector().

  4. ¿Cómo se modifica un elemento del DOM?

    Para modificar un elemento del DOM, se pueden cambiar sus propiedades como innerHTML, textContent, o modificar estilos mediante style.

  5. ¿Cuál es la diferencia entre getElementBy* y querySelector*?

    getElementBy* selecciona elementos basados en ID o clase, mientras que querySelector* usa selectores CSS, permitiendo más flexibilidad.

  6. ¿Qué es un evento?

    Un evento es una acción que ocurre en la página, como un clic o una pulsación de tecla, y puede ser manejado mediante JavaScript.

  7. ¿Qué es un listener?

    Un listener es una función que escucha un evento en un elemento y se ejecuta cuando dicho evento ocurre.

  8. ¿Qué es un callback?

    Un callback es una función pasada como argumento a otra función, que se ejecutará después de un evento o una operación asíncrona.

  9. ¿Qué es un objeto de evento?

    Un objeto de evento es un objeto que contiene información sobre un evento ocurrido, como el tipo de evento y el elemento que lo originó.
