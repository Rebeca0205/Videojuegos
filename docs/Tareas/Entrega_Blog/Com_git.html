Flujo de trabajo de git y github

¿Qué es git?

Git es un sistema de control de versiones que realiza un seguimiento de los cambios en los archivos. Git es especialmente útil cuando un grupo de personas y tú estás haciendo cambios en los mismos archivos al mismo tiempo.

¿Qué es github?

GitHub es una plataforma basada en la nube donde puedes almacenar, compartir y trabajar junto con otros usuarios para escribir código. Ahi se encontrará tu repositorio remoto

Comandos de git

Comando Descripcion
git status Revisa el estado de la carpeta actual y si se encuentran cambios
git --version Revisar la version actual de git que tiene tu dispositivo
git add 'archivo' o 'carpeta' Revisa archivos modificados o agregados y los agrega al staging area
git restore --staged 'archivo' Quita el archivo del staging area despues de haberlo añadido.
git restore 'archivo' Restaura el archivo a su ultima version antes de los cambios.
git commit -m "mensaje" Compromete todos los nuevos cambios y los registra en la base de datos de cambios, además que tambien manda un mensaje del cambio realizado
git push -u origin main Agarra los cambios y los sincroniza con el repositorio remoto de github
git clone 'link del ssh del repositorio' Crea un repositorio local ligado con el repositorio remoto de github
git branch -a Muestra todas las ramas del repositorio local y remoto
git branch 'rama' Crea una nueva rama
git checkout -b 'rama' Crea una nueva rama con el nombre dado y directamente de cambia a esa rama.
git checkout 'rama' , git switch 'rama' Te cambia a la rama escrita
git push -u origin 'rama' Hace un push desde la rama seleccionada, y todos los cambios se mantienen en esa rama
git merge 'rama' Merge la rama seleccionada con la rama principal para guardar los cambios. Este comando se debe de usar estando en la rama principal
git push Hace el push del merge
git branch -d 'rama' Borra la rama seleccionada
git push origin -d 'rama' Borra la rama seleccionada del repositorio remoto
git fetch -a Jala todas las ramas y los cambios hechos del repositorio remoto
git pull origin 'rama' Jala todos los cambios ya hechos hacia la rama seleccionada
