Conceptos de Css

¿Qué es Css?

Css quiere decir "Hojas de estilo en cascada", es decir, es un lenguaje de hojas de estilo que se utiliza para describir la presentación de un documento escrito en HTML o XML (incluidos los dialectos XML como SVG , MathML o XHTML ). CSS describe cómo deben representarse los elementos en la pantalla, en papel, en el habla o en otros medios.

Terminos de Css

Un selector designa exactamente a qué elemento o elementos de nuestro HTML se debe apuntar y aplicar estilos (como color, tamaño y posición). Los selectores pueden incluir una combinación de diferentes calificadores para seleccionar elementos únicos, todo dependiendo de cuán específicos queramos ser.

Una vez que se selecciona un elemento, una propiedad determina los estilos que se aplicarán a ese elemento. Los nombres de las propiedades se colocan después de un selector, dentro de las llaves, {}, e inmediatamente antes de dos puntos (:)

Finalmente, se determina el comportamiento de las propiedades con un valor. Los valores se pueden identificar como el texto entre los dos puntos (:) y el punto y coma (;)

Tipos de selectores

Los selectores, como se mencionó anteriormente, indican qué elementos HTML se van a diseñar. Es importante comprender completamente cómo se utilizan los selectores y cómo se pueden aprovechar. El primer paso es familiarizarse con los diferentes tipos de selectores: selectores de tipo, clase e ID.

Tipo

Los selectores de tipo seleccionan elementos según su tipo de elemento: div, p, section, article, etc. No son muy especificos ya que solamente toman del mismo tipo independientemente de su posicion en la pagina web.

div {...}

Clase

Los selectores de clase permiten seleccionar un elemento en función del valor clase del atributo del elemento. Estos selectores son un poco más específicos, ya que seleccionan un grupo particular de elementos en lugar de todos los elementos de un tipo. Los selectores de clase permiten aplicar los mismos estilos a diferentes elementos a la vez utilizando el mismo valor de clase de atributo en múltiples elementos. En CSS, las clases se indican con un punto inicial (.) seguido del valor clase del atributo.

.awesome {...}

ID

Los selectores de ID son incluso más precisos, ya que solo se dirigen a un elemento único a la vez. Ya que estos selectores utilizan un valor ID del atributo de un elemento como selector. Independientemente del tipo de elemento en el que aparezcan, los valores ID de atributo solo se pueden utilizar una vez por página. Si se utilizan, deben reservarse para elementos importantes. En CSS, los selectores de ID se indican con un signo de hashtag (#) seguido del valor ID del atributo.

#shawhowe {...}
